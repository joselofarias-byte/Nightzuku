# root-compatibility-check

Use when touching root logic.

Checklist:
- Root unavailable.
- Root denied.
- Root granted.
- Magisk present/absent.
- KernelSU present/absent.
- APatch present/absent.
- Shell command failure.
- Timeout behavior.
- Non-root fallback.
- Shizuku fallback.
- No destructive command.
- Clear user-visible error.

Output:
- Capability detection logic.
- Failure handling.
- Safe fallback.
- Validation commands.
