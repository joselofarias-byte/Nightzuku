# stellar-nightzuku-compatibility

Use when working specifically on Stellar or Nightzuku.

Checklist:
- Existing working apps remain compatible.
- Shizuku server startup remains stable.
- Service attach/reconnect remains stable.
- Root emulation does not overclaim capability.
- Shell command timeouts are safe.
- Diagnostics expose enough information.
- UI does not hide critical failure reason.
- Battery/OEM restrictions are considered.
- Android SDK 36 behavior is considered.

Output:
- Compatibility risks.
- Regression risks.
- Required manual tests.
- Safe rollout recommendation.
