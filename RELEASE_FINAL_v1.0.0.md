# Nightzuku Fork v1.0.0

## English

This is a stable release of my personal Nightzuku/Shizuku fork.

I am not the original developer of the project. This release is an adaptation focused on Spanish support, usability improvements, Android TV and Wear OS polish, and selected safe fixes.

### Highlights

- Spanish localization added and refined.
- Home service status loading improved.
- Android TV labels, icons, and loading states improved.
- Wear OS loading behavior and scroll transformations improved.
- Application Management improved across Phone, TV, and Wear.
- ADB pairing peer info fix integrated from upstream r49.
- Release build optimized with R8, resource shrinking, and debug/info/verbose log stripping.
- ADB pairing and Wireless Debugging validated on the maintainer device.

### Build

- Variant: release
- Signed APK: yes
- R8: enabled
- Resource shrinking: enabled
- APK size: about 3.6 MB

## Español

Esta es una versión estable de mi fork/adaptación personal de Nightzuku/Shizuku.

No soy el programador original del proyecto. Esta versión es una adaptación enfocada en soporte en español, mejoras de uso, pulido para Android TV y Wear OS, y algunos fixes seguros seleccionados.

### Puntos principales

- Soporte en español agregado y pulido.
- Mejoras en el estado de carga del servicio en Home.
- Mejoras en etiquetas, iconos y estados de carga para Android TV.
- Mejoras en carga y transformaciones visuales de scroll para Wear OS.
- Mejoras en Application Management en Phone, TV y Wear.
- Fix de ADB pairing peer info integrado desde upstream r49.
- Build release optimizado con R8, reducción de recursos y eliminación de logs debug/info/verbose.
- ADB pairing y Wireless Debugging validados en el dispositivo del mantenedor.

### Build

- Variante: release
- APK firmado: sí
- R8: activado
- Reducción de recursos: activada
- Tamaño APK: aproximadamente 3.6 MB
