# Nightzuku — Personal Adaptation Notice / Aviso de adaptación personal

## English

This repository is a personal fork/adaptation based on the original Nightzuku/Shizuku project. I am not the original developer of the software and I do not claim authorship of the original codebase.

My work focuses on adapting the application to my own needs, improving usability where possible, and adding Spanish support for users who may have difficulty using the application in English.

The goal of this fork is to preserve the main behavior of the original project while adding controlled improvements in user interface, accessibility, localization, loading states, Wear OS experience, Android TV experience, and small compatibility fixes when they can be applied safely.

Special care was taken to avoid unnecessary changes in sensitive areas such as server logic, binder / IPC behavior, authorization logic, permission granting or revoking behavior, package identity, rish, shell runtime behavior, automatic startup, foreground service behavior, and broad Manifest or Gradle changes.

Some improvements from later upstream versions were reviewed and integrated selectively, but upstream releases were not merged wholesale when doing so could risk removing or conflicting with changes already made in this fork.

The goal of this repository is not to replace the original project, but to provide a personalized adaptation with Spanish language support and practical UX improvements.

## Español

Este repositorio es un fork/adaptación personal basado en el proyecto original Nightzuku/Shizuku. No soy el programador original del software y no reclamo autoría sobre la base original del proyecto.

Mi trabajo se centra en adaptar la aplicación a mis propias necesidades, mejorar la experiencia de uso cuando sea posible y agregar soporte en español para usuarios que puedan tener dificultades usando la aplicación en inglés.

La intención de este fork es conservar el comportamiento principal del proyecto original, incorporando mejoras controladas en interfaz, accesibilidad, localización, estados de carga, experiencia en Wear OS, experiencia en Android TV y pequeños ajustes de compatibilidad cuando se pueden aplicar de forma segura.

Se tuvo especial cuidado para evitar cambios innecesarios en áreas sensibles como lógica del servidor, binder / IPC, lógica de autorización, otorgamiento o revocación de permisos, identidad del paquete, rish, shell runtime, arranque automático, foreground service y cambios amplios de Manifest o Gradle.

Algunas mejoras de versiones posteriores del upstream fueron revisadas e integradas de forma selectiva, pero no se hizo un merge completo de releases upstream cuando eso podía eliminar o entrar en conflicto con cambios ya realizados en este fork.

El objetivo de este repositorio no es reemplazar al proyecto original, sino ofrecer una adaptación personalizada con soporte en español y mejoras prácticas de experiencia de usuario.
